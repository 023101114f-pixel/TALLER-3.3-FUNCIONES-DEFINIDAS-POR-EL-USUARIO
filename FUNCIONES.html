UDF

10 Funciones Definidas por el Usuario

Bases: Northwind · Pubs — SQL Server

Introducción

Este proyecto contiene 10 funciones creadas en SQL Server (escalares y de tabla), diseñadas para resolver problemas reales como análisis de ventas, validación de datos, control editorial y reportes personalizados. Las funciones se aplican sobre Northwind y Pubs, integrando lógica reutilizable y optimizada.

Funciones — Northwind (7)

Tabla Inline

1) PedidosPorClienteAnio

Retorna pedidos por cliente y año. Útil para auditorías y reportes históricos.

Escalar

2) TotalVentasEmpleadoPeriodo

Calcula el total vendido por un empleado en un rango de fechas.

Tabla Inline

3) ProductosSinStock

Muestra los productos agotados o en nivel crítico.

Escalar

4) ClienteTienePedidos

Verifica si un cliente tiene registros de pedidos.

Multisentencia

5) TopProductosPorCategoria

Lista los productos más vendidos por categoría.

Tabla Inline

6) ProductosVendidosEnRango

Muestra productos vendidos entre dos fechas con importes totales.

Escalar

7) CantidadPedidosPais

Cuenta cuántos pedidos fueron enviados a un país específico.

Funciones — Pubs (3)

Tabla Inline

8) AutoresSinLibros

Identifica autores que no tienen títulos asignados.

Tabla Inline

9) TitulosNoVendidos

Detecta títulos que nunca registraron ventas.

Escalar

10) PrecioMedioPublisher

Calcula el precio promedio de los libros publicados por una editorial.

Conclusión

Estas UDFs facilitan la organización y análisis de datos, permitiendo automatizar cálculos, validar relaciones y generar reportes en distintos escenarios. El trabajo reforzó el uso de SQL Server para crear soluciones modulares, reutilizables y fáciles de integrar.
